The Cowenhoven Project — Data Tools

Recovering Black Lives from Enslaver Archives in New York and New Jersey

This is the data subdomain of cowenhoven.org. The project documents people of African descent enslaved in New York and New Jersey by the Van Kouwenhoven-Conover family. The tools below give different views of the underlying dataset.

Live Family Tree

An interactive descent tree from Wolphert Gerritsz van Couwenhoven (1579–1662) through nine generations, color-coded by branch and showing enslaved persons matched to each enslaver.

Project Infographic

A one-page visual overview of the project’s scope, dataset, and findings to date.

Index of Individuals

Every named individual identified in archival documents and public records, with filters by county, event type, age category, and date range.

Enslaver Cluster Map

A geographic map of the Cowenhoven enslaver clusters across New York and New Jersey, organized by branch and migration pattern.
